## 🚀 About Me

Cloud-/DevOps-Background (AWS, Terraform, Docker, K8s) — heute voll auf **angewandte AI** fokussiert:
Ich betreibe einen komplett **self-hosted AI-Stack** (lokale LLMs auf AMD-GPUs, Agents, MCP-Server, automatisierte Pipelines) und baue eine **AI-Beratung für KMUs, Handwerk und NGOs** auf — pragmatische Automatisierung für Organisationen, die sich keine Enterprise-Beratung leisten können.

- 🔭 **Aktuell:** Self-hosted AI-Workspace (Docker · Ollama · AMD ROCm · MCP · Agents)
- 🌱 **Lerne:** DevSecOps · SOC/SIEM · Zero Trust
- 🐧 **Daily Driver:** Arch Linux + Hyprland — Dotfiles für 4 Distros, Catppuccin everywhere
